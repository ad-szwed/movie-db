movie db

Essential Features

Documentation
Request URL HTTP Method
Return a list of ALL movies to the user /movies GET
Return data about a single movie by title to the user /movies/title GET
Return data about a genre (description) by type (e.g., “Thriller”) /movies/genres/type GET
Return data about a director (bio, birth year, death year) by name /movies/directors/name GET
Allow new users to register /users POST
Allow users to update their username /users/username PUT
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/username/movies/title POST
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed) /users/username/movies/title DELETE
Allow existing users to deregister (showing only a text that a user email has been removed) /users/username DELETE
